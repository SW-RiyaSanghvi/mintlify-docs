# Compress Small Image API

## Endpoint

**POST** `/compress-small-image`

## Description

Compresses small images up to 75 MB to 40 KB.

## Request

- **Method**: POST
- **Content-Type**: multipart/form-data
- **Parameters**:
  - `image`: The image file to be compressed.

## Response

- **200 OK**: Returns the compressed image.
- **400 Bad Request**: If the image is invalid or cannot be compressed.

## Example

### Request

```bash
curl -X POST "http://your-api-url/compress-small-image" \
-H "accept: application/json" \
-H "Content-Type: multipart/form-data" \
-F "image=@/path/to/your/small-image.jpg"
```

### Response

```json
{
  "status": "success",
  "compressed_size": "40 KB"
}
```
