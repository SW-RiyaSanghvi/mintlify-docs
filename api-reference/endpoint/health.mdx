# Health Check API

## Endpoint

**GET** `/health`

## Description

Checks the health status of the service.

## Request

- **Method**: GET

## Response

- **200 OK**: Returns the health status.

## Example

### Request

```bash
curl -X GET "http://your-api-url/health" \
-H "accept: application/json"
```

### Response

```json
{
  "status": "healthy"
}
```
